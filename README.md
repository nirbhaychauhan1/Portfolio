## Deploy live for free (GitHub Pages)

1. **Push this repo to your GitHub** (e.g. repo name: `Portfolio` or `portfolio`).

2. **Turn on GitHub Pages**
   - Repo → **Settings** → **Pages**
   - Under **Build and deployment**, set **Source** to **GitHub Actions**.

3. **Run the workflow**
   - Push to the `main` branch, or go to **Actions** → **Deploy to GitHub Pages** → **Run workflow**.

4. **Your live site**
   - Project site: `https://<your-username>.github.io/<repo-name>/`  
     Example: `https://Nirbhay5634.github.io/Portfolio/`
   - User site (root URL): create a repo named `<username>.github.io`, push the same code, and the workflow will deploy to `https://<your-username>.github.io/`.
